Hi, we are SalezTalk.

We believe every website visitor is a potential customer. We also believe that for small businesses, the best way to increase sales is by talking to the customer.

SalezTalk is one of the simplest business tool for increasing sales call request from your website visitors. 

= WHAT DOES THE PLUGIN DO?  =

It places a call now widget (click to call) at the bottom of your website screen & helps you grow your business by enabling you to call back your customers faster.

== Installation ==

1. Create your account on SalezTalk.
2. Install the SalezTalk plugin either via the WordPress plugin directory, or by uploading the files to your web server (in the /wp-content/plugins/ directory).
3. Activate the plugin through the 'Plugins' menu in WordPress.
4. Visit SalezTalk dashboard and get the code. Now you need to paste this code in your WordPress SalezTalk menu (Under Settings) screen.


== Frequently Asked Questions ==

= Can I disable receiving call requests at night? =

Yes, you can set your timings in SalezTalk admin panel to disable receiving call requests at night or even your non working hours.

= Does this plugin slow down my site? =

Not at all. This plugin loads only a single JavaScript file so it wont slow down your site at all. Also the file is loaded asynchronously at bottom of your page so it will not affect your page speed.
